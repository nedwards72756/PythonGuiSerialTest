Introduction

The Scoreboard Serial Format Test program is provied to test scoreboard funcitonality against a list of known formats. Tests include correctly and incorrectly formated data. Each test will step thorough a list of test data strings. The data string will be tested for 2 seconds while sending out at 4 times a second.

Currently supported formats:

  • SMA
  • SB400
  • SB200
  • IQ355

Installation

Un-zip to directory you want the application to reside in.

Setup

Before launching program have all serial ports populated. If serial port does not show up in Com menu close application and re-open.

Usage

The top half of the application allow the user to select the COM port, Baud rate, and Start/Stop the testing proccess.

Each format has an assocated tab: SMA, SB400, SB200, and IQ355. Under each tab there are a differing number of test that can be performed.

Select the tab of the format you want to test. Check the test you want to run then press the Start Test button. As the test runs the bottom terminal will tell you what data string is being sent. Each data string is tested for 2 seconds and is sent at 4 times a second. When a test is complete it begins running again from the first test data string. When you are done press the Stop Test button.

Changes to checkboxes during an active test will not take effect untill the test is stopped, then started again.

Notice, when a test is ran the terminal will output the data string being tested. IF that data string is invalid the line will show up red and there will be an explanation of what the error is and what action the scoreboard should take.

Terminal Messages

  • Should not display:
    What ever the error is the scoreboard should be blank. No information from this data string should be used.
  • All digits cannot be displayed:
    More than 6 displayable digits are in the data string.
  • No room for negative sign:
    Data string has 6 digits to display and no room for the negative sign.
  • Gross/Net should not display but weight should still show:
    This data string should be displayed but a space has been sent where the gross/net data should be. A space means not to display Gross or Net but we still show weight.
  • Unkown gross/net:
    If the gross/net part of the data string is unknown or invalid we cannot show the weight.
  • Should display "O CAP:
    When we have an over capacity error we must display O CAP.
  • Unknown unit of measure:
    If a data string has an unkown unit of measure then the weight is not valid and cannot be displayed.
  • Other error:
    Other misc errors fall under this.
